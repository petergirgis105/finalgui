# Psychology Research Pool GUI

🔗 **Live site**: `https://petergirgis105.github.io/psychology-research-pool-gui/`

A fully interactive database GUI that runs entirely in the browser. Data is saved to browser storage (`localStorage`) so it survives page reloads.

## 🚀 Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. **Source**: `Deploy from a branch` → `main` → `/ (root)`
4. Click **Save**
5. Wait 30 seconds, visit your live URL

## 📦 Files

- `index.html` — The entire app (data + UI + logic in one file)

## 💾 Data Persistence

- Data saves automatically to browser `localStorage`
- Survives page reloads on the same device/browser
- Use **Export** button to download a `.json` backup
- Use **Import** button to restore from a `.json` file
- **Reset** button restores original seed data

## 🗃️ Tables

- Participants (25 seeded)
- Researchers (10 seeded)
- Ethics Approvals (10 seeded)
- Studies (10 seeded)
- Assignments (43 seeded)
- Credit Hours Log (29 seeded)
